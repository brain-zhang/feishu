<br>About feishu
<br>----------------
<br>python + webpy搭建的小站，用于对优酷的视频脱水。目前运行在sae上:
<br>http://feishu.sinaapp.com
<br>同样支持dotcloud。
<br>习作，代码很丑。
<br>

<br>        不支持列表播放页面的抓取
<br>        速度不是很理想
<br>        算法过于依赖页面结构，优酷改版的话可能会造成影响
<br>
<br>        优化中:
<br>
<br>        采用html结构解析的方式来分析
